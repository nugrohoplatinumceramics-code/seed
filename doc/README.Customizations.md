# SeedDMS Customizations

Dokumen ini disediakan untuk mencatat modifikasi kustom pada proyek SeedDMS ini.

## Tujuan

- Menjadi sumber referensi tunggal untuk semua perubahan lokal/kustom.
- Memudahkan proses upgrade versi SeedDMS di masa depan.
- Membantu proses audit ketika ada bug atau regresi.

## Template Pencatatan Modifikasi

Gunakan format berikut setiap kali menambahkan perubahan baru:

```md
### [Tanggal] Judul perubahan singkat
- **Lokasi file**: `path/ke/file`
- **Ringkasan perubahan**: Apa yang diubah.
- **Alasan bisnis/teknis**: Mengapa perubahan ini diperlukan.
- **Dampak**: Modul atau fitur yang terpengaruh.
- **Catatan upgrade**: Risiko konflik saat merge dengan rilis upstream.
```
